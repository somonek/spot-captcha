Implementation
Step 1:

Include jQuery plugin in your file. Supported versions 1.6+. Include jquery.spotsCaptcha.js in your file.

Step 2:

Create a container and append the captcha as in the example:

<div id="captcha-container"></div>
<script>
    var check = $("#captcha-container").spotsCaptcha();
</script>
Result:

Step 3:

Check the result.

if(check.getCaptchaResult()){
    //proceed
}
